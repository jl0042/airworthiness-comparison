# 适航标准对比 — eVTOL / 不载人无人机 vs 成熟有人机

> **在线访问**: <https://jl0042.github.io/airworthiness-comparison/>

中国民航局 2026 年 4 月颁布两份正常类不载人无人机适航标准 (UMR 多旋翼 / UPL 动力提升), 给无人机适航第一次有了跟 CCAR-23/27 同体系的通用基线。这个仓库把这两份新标准跟成熟有人机基线 (CCAR-23/25/27/29) 和同期还在跑的几条平行轨道 (V2000CG / EH216-S 专用条件 / 高风险货运试行) 做条款级对照, 169 条监管议题逐条对齐, 看监管思路从「保护舱内乘员」切到「保护地面第三方」之后, 删了什么, 加了什么, 改了什么。

## 一个主问题

针对 eVTOL / 不载人无人机的适航要求, 跟成熟有人机比哪些更严哪些更松, 这反映出监管思路怎么变了。

## 几条核心差异

- **整章删除 13 条**: 撤离设施 / 应急出口 / 乘员物理环境等 — 给舱内人用的条款在不载人无人机里整章删除。
- **全新增 101 条**: 伞降系统 / 鸟撞 / 外挂物 / 外部货物载荷; DAA + 电子围栏; C2 链路 5 条; 地面站 R 章 21 条; 电推进 H + I 章 64 条 — 监管把电推进从「正常类飞机的补充章」升格为「无人机的主章」。
- **方式转换 16 条**: 失速速度 → 飞行剖面 / 最小安全速度 (23.2110 ↔ UMR.2110 ↔ UPL.2110); 23.2270 应急情况 → UMR.2270 有效载荷舱 / UPL.2270 储存舱 — 同一个编号位置, 但说的事情已经不同。
- **38 条章节一一对应**: 主要在结构 / 飞行性能维度, 编号 / 标题对齐, 数字差异嵌在条款内部 (例: 极限载荷 = 限制载荷 × 1.5 三套标准都保留, 但 UPL.2230 多一句「经局方批准可使用 1.25-1.5 的安全系数」)。

## 数据来源

所有条款编号 / 标题 / 页码 / 原文均来自 `适航标准/` 目录下 12 份 CAAC 法规 PDF (10 份主对比标准 + CCAR-21 合格审定框架 + AC-92 中型无人机标准, 后两份用于分级矩阵和先例对照节), 没有任何第三方解读。

`data/` 目录:

- `airworthiness_mapping.csv` — 169 条主对比 mapping (议题 / 跨标准条款编号 / 页码 / 差异类型)
- `clause_bodies.json` — 1144 条条款的首段摘要 + 完整原文, 从 PDF 抽取, inline 嵌入 HTML
- `clauses_*.csv` — 各份 PDF 的单文件条款编号 / 标题 / 页码
- `source_audit.md` — 数据源审计, 事实陈述全部追到 PDF 原文 + 页码

`适航标准/` 目录: 10 份 CAAC PDF 原文。
